# AutoRegressive-ভাষা

**Language models go brrr.** A scrappy collection of notebooks where we poke, prod, and sweet-talk LLMs into doing cool things.

## What's Inside

```
google/          ~ Gemini & Vertex AI experiments
huggingface/     ~ Qwen explorations & dataset wrangling
llmware/         ~ SLIM model classification tricks
```

### `google/`
| Notebook | The Vibe |
|----------|----------|
| `Gemini_First_Contact.ipynb` | Multimodal shenanigans with Gemini Pro Vision |
| `Vertex_AI_prompting.ipynb` | Text generation via Vertex AI's Text Bison |

### `huggingface/`
| Notebook | The Vibe |
|----------|----------|
| `qwen_first-contact-001.ipynb` | First date with Qwen 2.5 (0.5B instruct) |
| `[hf]-simple_dataset_reformatter.ipynb` | Reshaping caption data for AutoTrain fine-tuning |

### `llmware/`
| Notebook | The Vibe |
|----------|----------|
| `SLIM_Category_Classification.ipynb` | Zero-to-category classification with SLIM |

## Quick Start

All notebooks are built for **Google Colab** -- click, run, explore. No local setup drama.
